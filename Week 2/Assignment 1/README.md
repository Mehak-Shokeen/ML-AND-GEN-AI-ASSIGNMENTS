# Netflix User Analytics using Machine Learning

## Project Overview
This project analyzes Netflix user data to understand customer behavior and apply Machine Learning techniques for predicting subscription renewal. The dataset contains information about 750 users, including demographics, viewing habits, subscription details, and spending patterns.

The goal is to identify factors influencing customer retention and support data-driven business decisions.

---

## Business Objectives

- Analyze user demographics and viewing patterns.
- Identify factors affecting subscription renewal.
- Predict whether a user will renew their subscription.
- Discover spending and engagement trends.
- Provide insights to improve customer retention.

---

## Dataset Information

The dataset contains 750 user records with the following attributes:

| Column Name | Description |
|------------|-------------|
| UserID | Unique user identifier |
| Age | User age |
| Gender | Male/Female |
| SubscriptionType | Basic, Premium, or VIP |
| WatchHoursPerWeek | Total hours watched per week |
| DevicesUsed | Number of devices used |
| FavoriteGenre | Preferred content genre |
| AdClicks | Number of advertisement clicks |
| MonthlySpend | Monthly spending (₹) |
| SubscriptionRenewed | Whether the subscription was renewed (Yes/No) |
